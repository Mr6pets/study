####	高级面向对象

- 包装对象
  - js基于原型的程序
  - string Number Boolean
- 原型链
  - 实例对象和原型之间的连接，叫做原型链
  - ____proto____（隐式链接）
  - Object对象类型是原型链的最外层
- 面向对象的一些属性和方法
  - hasOwnProperty()看是不是对象自生下面的属性
  - constructor:查看对象的构造函数
    - 每个原型都会自动添加constructor属性
    - for in 的时候有些属性是找不到的
    - 避免修改constructor属性
  - instanceof：运算符
    - 对象与构造函数在原型链上是否有关系
  - -toString()：object上的方法
